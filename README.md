# TicketApp

### The goal of this project is to create a bus travel app.

## Features
- Onboarding screen
- MapKit
- Search Feature
- Table View Cell and Collection View Cell
- Selecting Seats but layout must be fixed.
- Purchase Process
- The final screen is the ticket that user bought.
